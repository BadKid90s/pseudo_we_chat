# Flutter版微信（WeChat）V2.0

使用Flutter框架开发微信页面

## 版本

### 当前版本[V2.0版本](README.md)

### 历史版本
- [V1.0版本](README_V1.0.md)

## 特点
 - 支持国际化（中文、英文）
 - 支持主题自适应（浅色/深色）
 - 支持对接后台进行真实接口对接
 - 使用GetX框架
 - 使用Dio进行数据交互
 - 使用getwidget做为UI库
